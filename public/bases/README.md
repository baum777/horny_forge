# Base Images

Place the following 4 PNG images in this directory:

1. `base-01-unicorn-head.png` - Core Head (Tight Brand)
2. `base-02-landscape.png` - Landscape (Lore Scene)
3. `base-03-jesus-meme.png` - Holy Ascension (Epic Meme)
4. `base-04-rocket.png` - Rocket (Meta Launch)

These are the frozen base images used by the HORNY META FORGE. The server maps base_id to these actual image files.
